# english_study
Train your ability of speaking or listening or translation ability
这个小工具就针对对英语还有那么一点点兴趣，想脱口说英语，但又不知道如何表达。学习时间是碎片化，没有大把时间专门学习的小主儿。 学习的形式就以汉译英的形式答题，可以根据不同级别的难度（比如雅思，托福，英语6级），随机根据难度来出题。 题目为中文的一句话，根据答案的单词的个数，出现对应的空缺，要求答题者填写，最终形成一个完整的句子。随测随练，方便快捷，实在不会翻译还可以先听一遍英语，顺便练习了听力，连续答对3道题，还有彩蛋哟！
